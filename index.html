About This Project

This website is automatically deployed to Amazon S3 using GitHub Actions. After deployment, CloudFront is invalidated so the latest version is served through the CDN.

What This Shows

  • Using GitHub Actions for CI/CD
  • Uploading website files to Amazon S3 automatically
  • Invalidating CloudFront cache after deployment
  • Using AWS IAM permissions for deployment automation
  • Building a real-world static website deployment workflow
